# 介绍

agility-form 是一款面向 PC 端中后台的表单渲染组件解决方案。

对于 PC 后台，有大量简单重复页面，为了更高效开发，我们基于 JSON 配置的方式动态生成 UI，包含查询表单、表格等；

agility-form 包含组件：agilitySearchForm(查询表单)、agilityTable（表格）
